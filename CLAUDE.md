# Product: Instructor Capacity Scheduling System

## Product Overview
This product is a lightweight internal scheduling tool designed for vocational training schools (such as CDL schools). The system helps managers assign students to instructors without exceeding instructor capacity.

The system replaces manual scheduling coordination currently happening across CRMs, spreadsheets, Google Drive, texting, and phone calls.

The MVP focuses on one core function:
Assign students to instructors while preventing instructor overload.

All students entered into the system are assumed to be ready for scheduling. Student readiness validation will remain outside the system for the MVP.

The system acts as an operational visibility tool rather than replacing existing CRMs or student management systems.

## Target User
Owner-operators or managers of small vocational training schools.

Typical characteristics:
- 3–8 instructors
- 20–60 active students
- Personally responsible for scheduling and coordination
- Currently managing scheduling manually

Example user:
Adrian Choquevillca – owner/operator of a CDL training school.

## Value Proposition
We help vocational school managers eliminate scheduling chaos by making instructor capacity and student assignments visible in one simple system.

## Tech Stack (MVP)
Frontend: Next.js / React  
Backend: Next.js API routes  
Database: PostgreSQL (Supabase recommended)  
Hosting: Vercel  
AI Development Assistant: Claude

The system should remain simple, internal, and easily modifiable.
